# Troubleshooting & Common Fixes
If a smart device is offline:
1. Check Wi-Fi network and signal strength.
2. Restart the router and the device.
3. Ensure the device firmware is updated.
4. Reset the device to factory settings (use only if instructed).

If app login fails:
- Verify your email/phone and password.
- Reset password via 'Forgot Password' link.
- Clear app cache and update the app.
